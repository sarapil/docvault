## Design Guidelines

- All screens use `frappe_visual` components
- Minimum 3 `.fv-fx-*` CSS effects per screen
- CSS Logical Properties for RTL (no margin-left/right)
- Icons from `frappe.visual.icons` only
- All strings wrapped in `__()` for i18n
- Responsive: 320px to 4K
