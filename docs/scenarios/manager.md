# DV Manager — Usage Scenarios
# سيناريوهات استخدام — مدير المستندات

## Role Overview

- **Title**: Document Manager / مدير المستندات
- **CAPS Capabilities**: DV_manage_workflows, DV_approve_documents, DV_manage_sharing, DV_view_audit_trail
- **Primary DocTypes**: DV Review Request, DV Approval, DV Share, DV Audit Trail
- **Device**: Desktop

## Daily Scenarios (يومي)

### DS-001: Review Pending Approvals
- **Goal**: Process document review requests
- **Steps**:
  1. Navigate to DV Workflows workspace
  2. Filter DV Review Request by status = "Pending"
  3. Open document, review content
  4. Approve or reject with comments

### DS-002: Share Document with Team
- **Goal**: Grant document access to specific users
- **Steps**:
  1. Open DV Document form
  2. Click Share → DV Share → New
  3. Set user, permission level (view/edit/download)
  4. Save — user receives notification

## Weekly Scenarios (أسبوعي)

### WS-001: Audit Document Activity
- **Goal**: Review who accessed what documents
- **Steps**:
  1. Navigate to DV Compliance → DV Audit Trail
  2. Filter by date range and action type
  3. Flag unusual access patterns
