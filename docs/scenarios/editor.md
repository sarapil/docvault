# DV Editor — Usage Scenarios

# سيناريوهات استخدام — محرر المستندات

## Role Overview

- **Title**: Document Editor / محرر المستندات
- **CAPS Capabilities**: DV_create_documents, DV_edit_documents, DV_manage_versions, DV_checkout_documents, DV_manage_tags
- **Primary DocTypes**: DV Document, DV Version, DV Checkout, DV Tag
- **Device**: Desktop / Tablet

## Daily Scenarios (يومي)

### DS-001: Upload New Document

- **Goal**: Upload and classify a new document
- **Steps**:
  1. Navigate to DV Core → DV Document → New
  2. Set title, upload file, select category and folder
  3. Add tags for searchability
  4. Save — version 1 created automatically

### DS-002: Upload New Version

- **Goal**: Update an existing document with a new file
- **Steps**:
  1. Open existing DV Document
  2. Check out document (locks for exclusive edit)
  3. Upload new file via version upload
  4. Add change notes
  5. Check in — version incremented

### DS-003: Search for Documents

- **Goal**: Find a specific document
- **Steps**:
  1. Navigate to DV Search workspace
  2. Enter search query (title, keywords, tags)
  3. Filter by category or folder
  4. Open matching document
