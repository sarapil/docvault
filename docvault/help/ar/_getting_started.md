# البدء

1. اذهب إلى **إعدادات DV** لتكوين التخزين والإصدارات والتعرف الضوئي
2. أنشئ إدخالات **فئة DV** لتنظيم مستنداتك
3. أنشئ تسلسل **مجلد DV** للتنظيم المنطقي
4. ارفع المستندات عبر **مستند DV** — يتم تتبع الإصدارات تلقائياً
5. أعد **سياسة احتفاظ DV** لمتطلبات الامتثال
6. كوّن **مزود تخزين DV** للتخزين الخارجي
