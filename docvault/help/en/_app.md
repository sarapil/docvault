# DocVault — Enterprise Document Management System
DocVault provides comprehensive document management with version control, approval workflows, OCR, full-text search, compliance tracking, and external storage integration.

## Key Features
- **Version Control**: Full version history with checkout/checkin
- **Approval Workflows**: Multi-level review and approval
- **OCR**: Extract text from scanned documents
- **Full-Text Search**: Find documents by content
- **Compliance**: Retention policies, audit trails, controlled documents
- **External Storage**: S3, Nextcloud, Google Drive, Azure integration
- **Sharing**: Secure share links with expiry and password protection
