# Getting Started

1. Go to **DV Settings** to configure storage, versioning, and OCR
2. Create **DV Category** entries to organize your documents
3. Create **DV Folder** hierarchy for logical organization
4. Upload documents via **DV Document** — versions are tracked automatically
5. Set up **DV Retention Policy** for compliance requirements
6. Configure **DV Storage Provider** for external storage (S3, Nextcloud)
